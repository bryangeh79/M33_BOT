# 项目概览

## 项目名称：
M33 Lotto Bot

## 项目路径：
C:\M33 Lotto Bot

## 项目目标：
M33 Lotto Bot 是一个 Telegram 自动化彩票系统，未来将支持：

- 开奖数据抓取
- 用户查询开奖
- 下注记录管理
- 自动结算系统
- 管理员控制
- 日志与监控

## 系统范围：
该系统将涵盖以下功能模块：
- 开奖数据展示
- 用户下注管理

## 不在范围：
- 涉及法律的彩票活动
- 与具体彩票公司直接的集成

## 项目角色：
- Product Owner
- Engineering Supervisor
- AI Coding Agent
- Operator

## 项目用途：
为参与乐透游戏的用户提供便利的开奖信息。

## 目标用户：
对乐透感兴趣的玩家。

## 核心功能：
- 提供实时的开奖信息
- 支持用户查询历史开奖信息
- 其他功能可扩展（如用户自定义提醒等）

## 当前阶段：

## 当前已知情况：
项目框架正在建设中，核心功能尚待定义。
