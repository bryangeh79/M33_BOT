# M33 Lotto Bot Installation Instructions

## 1. Ensure Node.js and npm are installed
   - Check versions:
     ```bash
     node -v
     npm -v
     ```

## 2. Install dependencies
   - Navigate to the project folder:
     ```bash
     cd C:\testing-boy-bot
     ```
   - Install Telegraf:
     ```bash
     npm install telegraf
     ```
   - Install other dependencies:
     ```bash
     npm install axios cheerio
     ```

## 3. Start the bot
   - Create the main `index.js` file to initialize the bot and handle commands.

## 4. Directory Structure
   ```plaintext
   M33 Lotto Bot/
   ├── actions/
   ├── services/
   ├── rules/
   ├── utils/
   ├── data/
   ├── scripts/
   ├── INSTALL.md
   └── index.js
   ```
